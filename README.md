```bash

npm install

# Start development server
npm run dev
```

## Adding Shadcn UI Components

To add Shadcn UI components to your project:

```bash
npx shadcn@latest add [component-name]
```

Example:
```bash
npx shadcn@latest add button
npx shadcn@latest add card
```
